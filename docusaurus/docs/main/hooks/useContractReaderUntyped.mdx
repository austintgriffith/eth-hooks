Call a contract function with arguments and receive the output.

You can use this to easily keep track of contract outputs in react states.

> Returns undefined or output.

## How to import

```typescript
import { useContractReaderUntyped } from 'eth-hooks';
```

## How to use

To get started:

```typescript
const readContract = useContractReaderUntyped<GOutput>(contract, contractFunctionInfo, formatter?, onChange?, override?): undefined | GOutput
```
