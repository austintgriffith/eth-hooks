Get the Uniswap exchange price of ETH/USD (extrapolated from WETH/DAI).

## How to import

```typescript
import { useDexEthPrice } from 'eth-hooks';
```

## How to use

To get started:

```typescript
const price = useDexEthPrice();
```

[Go here for this hooks full API documentation](../../api/modules/Hooks#usedexethprice).
