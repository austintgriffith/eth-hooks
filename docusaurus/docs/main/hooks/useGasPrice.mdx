Get the price of Gas from EtherGasStation ⛽️

## How to import

```typescript
import { useGasPrice } from 'eth-hooks';
```

## How to use

To get started:

```typescript
const [gasPrice] = useGasPrice(ethersContext.chainId, 'fast', getNetworkInfo(ethersContext.chainId));
```

[Go here for this hooks full API documentation](../../api/modules/Hooks#usegasprice).
