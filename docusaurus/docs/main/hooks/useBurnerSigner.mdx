Creates a burner signer/address, and provides ways of interacting with and updating the signer

## How to import

```typescript
import { useBurnerSigner } from 'eth-hooks';
```

## How to use

To get started:

```typescript
const burnerSigner = useBurnerSigner();
```

[Go here for this hooks full API documentation](../../api/modules/Hooks#useburnersigner).
