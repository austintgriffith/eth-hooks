Get the balance of an ERC20 token at an address.

> Returns undefined or the balance.

## How to import

```typescript
import { useTokenBalance } from 'eth-hooks';
```

## How to use

To get started:

```typescript
const balance = useTokenBalance<GContract>(contract, address, options?): THookResult<BigNumber>
```
