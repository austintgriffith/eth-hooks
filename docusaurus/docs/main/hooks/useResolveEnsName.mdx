Gets ENS name form a given wallet address.

> Returns undefined or the ENS.

## How to import

```typescript
import { useResolveEnsName } from 'eth-hooks';
```

## How to use

To get started:

```typescript
const ensName = useResolveEnsName(mainnetProvider, address): THookResult<undefined | string>
```
