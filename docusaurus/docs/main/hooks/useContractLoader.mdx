Load your contracts and return them.

> Returns a record of the contract name.

## How to import

```typescript
import { useContractLoader } from 'eth-hooks';
```

## How to use

To get started:

```typescript
const loadContract = useContractLoader(config?, providerOrSigner): Record<string, BaseContract>;
```
