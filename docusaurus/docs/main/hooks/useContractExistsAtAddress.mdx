Find out if a contract exists already on the blockchain.

> Returns a boolean.

## How to import

```typescript
import { useContractExistsAtAddress } from 'eth-hooks';
```

## How to use

To get started:

```typescript
const contractExists = useContractExistsAtAddress(contract, options?): THookResult<boolean>
```
