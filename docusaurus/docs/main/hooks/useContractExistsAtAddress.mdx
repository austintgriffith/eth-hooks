Find out if a contract exists already on the blockchain.

## How to import

```typescript
import { useContractExistsAtAddress } from 'eth-hooks';
```

## How to use

To get started:

```typescript
const contractExists = useContractExistsAtAddress();
```

[Go here for this hooks full API documentation](../../api/modules/Hooks#usecontractexistsataddress).
