Get a token list from Uniswap's IPFS token list.

> Returns a token list

## How to import

```typescript
import { useDexTokenList } from 'eth-hooks';
```

## How to use

To get started:

```typescript
const tokenList = useDexTokenList(tokenListUri?, chainId?, options?): THookResult<TokenInfo[]>
```
